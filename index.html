ViCaRe — VISSIM Calibrator and Reporter

Role: Developer, Data Scientist, and Product Designer

ViCaRe is a professional automation software for VISSIM calibration, simulation result extraction, data processing, and Excel reporting. It automates workflows that traditionally require many hours of repetitive manual work.

  • Automated reporting for nodes, links, lanes, travel time, throughput, and network-level results.
  • Designed calibration workflows using genetic algorithms and multi-scenario parameter search.
  • Built repeatable data-processing workflows for transportation simulation projects.
  • Converted complex simulation outputs into structured, formatted, decision-ready Excel reports.
